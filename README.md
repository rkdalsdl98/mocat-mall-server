어드민 코드를 payload에 담아 발행  

가드에서 어드민 코드 복호화 => 서비스에 넘김 => 서비스에서 접근 권한이 필요한 요청이라면 복호화된 코드를 ":"로 스플릿 (0: 접근 권한, 1: 권한 코드) => 권한 코드를 유저가 가지고있는 권한 테이블에 salt로 원본 hash와 비교 => 접근권한이 요구 권한에 충족할 경우 허용
